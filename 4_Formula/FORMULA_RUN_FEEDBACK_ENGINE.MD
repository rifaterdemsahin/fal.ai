### How to use it:

1. The script will show you a file path.
2. Type your feedback and hit  **Enter** .
3. If you have no feedback for a file, just hit **Enter** to skip it.
4. Every  **5 feedback items** , it will save a batch file (e.g.,
   ```
   feedback_batches/repo_fix_batch_1.yaml
   ```
   ) in the folder where you ran the script.
